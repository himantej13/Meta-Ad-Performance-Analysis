#  Meta Ad Performance Dashboard

This Power BI project visualizes and analyzes Meta (Facebook & Instagram) ad campaign performance to evaluate marketing effectiveness and ROI.

---

##  Project Overview
The dashboard provides insights into ad metrics such as:
- **Impressions, Clicks, CTR, Engagement Rate, Conversion Rate, and Purchase Rate**
- **Gender and Age-based Engagement**
- **Geographic Engagement Distribution**
- **Weekly & Hourly Engagement Trends**
- **Ad Type Analysis (Carousel, Image, Stories, Video)**
- **Budget Utilization and Campaign Performance**

---

##  Key Insights
- Overall **CTR (Click Through Rate): 11.76%**
- Highest engagement rate observed among **female users (43%)**
- Majority engagement from **age group 20–35**
- **Stories** ads achieved the best conversion and engagement performance
- **Total Ad Budget:** $2.54M across multiple campaigns
